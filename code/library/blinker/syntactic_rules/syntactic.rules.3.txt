
# Rule 3: MAIN_EVENT - MAIN_EVENT
# ===============================
#
# Applies to:
# - Two EVENTS that are both the first event in consecutive sentences

# Always set:
#
#  - conjBetweenEvents=NO
#  - arg1.firstVbEvent=YES   
#  - arg2.firstVbEvent=YES
#
#    i.e. both events are main events in their respective sentences
#
